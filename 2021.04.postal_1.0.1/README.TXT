INSTALLATION
1. Copy the contents of the Postal menu.py file to your menu.py file located in your .nuke directory
2. Copy the contents of the Postal init.py file to your init.py file located in your .nuke directory
3. Place the Postal_1.0.1 folder into your .nuke directory
4. Launch Nuke and press F9 with a node selected to create a parent node

For more information, watch the demo video and read the blog post for this tool on my website
https://www.isaacspiegel.com/blog/keep-your-node-graph-tidy-with-postal-an-input-connection-manger-for-nuke
